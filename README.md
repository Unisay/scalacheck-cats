[Cats](http://typelevel.org/cats) typeclass instances for ScalaCheck data types

# Usage #
```sbt
libraryDependencies += "io.github.amrhassan" %% "scalacheck-cats" % "0.1.2" % Test
```
```scala
import scalacheck.cats._
```
